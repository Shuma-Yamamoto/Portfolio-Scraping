# Portfolio-Scraping

## ①概要説明
某求人サイトに掲載されている求人情報を取得し、CSV形式で保存するためのコードです。
<br><br>
取得した求人情報
<br><br>
※当該コードはインターンシップで制作したものを基に、求人サイトの特定を防ぐ形でコードを一部改変しています。

## ②使用環境
アーキテクチャは以下の通りです。
<br>

<br>
EC2-Amazon Linux 2
<br>
Python-3.7

## ③使用方法
１．アーキテクチャに従ってEC2を立ち上げる。
<br>
２．コンソール画面を起動し、当該リポジトリをクローンする。
<br>
３．必要なライブラリをインストールする。
<br>
　　(※仮想環境へのインストールを推奨)
<br>
４．app.pyを実行する。
